1.  Реализовать клиентское приложение чата с использованием RMI
1.1 Написать реализацию ChatLocalNodeImpl интерфейса IChatLocalNode, которая позволяет принимать сообщения удалённо,
путём вызова метода IChatLocalNode.receive() и накапливать в каком-нибудь хранилище, например, в очереди.
1.2 Создать класс ChatClient с методом main()
1.3 Локальный RMI registry следует поднимать на порте 1099
1.4 При запуске приложения следует экспортировать ChatLocalNodeImpl с портом 10000, инициализировать skeleton
IChatLocalNode и зарегистрировать в локальном RMI registry по имени "localNode"
1.5 Получить stub IChatCentralNode из удалённого RMI registry по имени "centralNode"
1.6 Зарегистрировать клиента в чате путём удалённого вызова IChatCentralNode.register() с указанием порта локального
registry для обратной связи и никнейма
1.7 Реализовать вычитку накопленных в реализации IChatLocalNode сообщений, например, путём вывода в System.out в
выделеном потоке
1.8 Реализовать возможность ввода сообщений и отправки на сервер чата путём удалённого вызова IChatCentralNode.send()
2. Реализовать серверное приложение чата с использованием RMI
2.1 Написать реализовацию ChatCentralNodeImpl интерфейса IChatCentralNode, которая позволяет регистрировать портебителей
по вызову метода IChatCentralNode.register() и пересылать сообщение, полученное от потребиеля в методе IChatCentralNode.receive()
2.2 Локальный RMI registry следует поднимать на порте 1099
IChatCentralNode.receive() всем зарегистрированм пользователям с соответствующим форматированием время-никнэм-сообщение.
2.3 Создать класс ChatServer с методом main()
2.4 При запуске приложения следует инициализировать skeleton IChatCentralNode, и зарегистрировать в локальном RMI registry
по имени "centralNode"
